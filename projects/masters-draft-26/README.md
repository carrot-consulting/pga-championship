# Masters Draft 2026

A single-page web app for a 6-person Masters sweepstake. Each player drafts 4 golfers before the tournament. The leaderboard shows combined scores, auto-refreshes every 10 minutes, and supports a Sunday captain multiplier mechanic.

**Status:** Building

**Key dates:**
- Draft picks: Wednesday April 8
- Tournament: April 9–13, 2026
- Captains set: Saturday April 12 (before R4)

**Hosted at:** GitHub Pages (new repo: `masters-draft-26`)

## Repo

Code lives in `repo/` — push that folder directly to the `masters-draft-26` GitHub repo.

## Testing

Using Valero Texas Open (current ESPN live leaderboard) with dummy picks to validate API integration before Masters week.
